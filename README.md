# Толмач — живой переводчик переговоров (китайский ↔ русский)

Android-приложение для деловых встреч с китайскими партнёрами. Телефон лежит на столе, вы — в Bluetooth-наушниках.

**Как это работает на встрече:**

1. Нажимаете «Слушать китайский». Партнёр говорит — телефон распознаёт речь, переводит и тихо озвучивает русский перевод **вам в наушники**. Собеседник ничего не слышит.
2. Хотите ответить — нажимаете «Ответить по-русски» и говорите. Телефон озвучивает вашу фразу **по-китайски через динамик** — её слышит партнёр. После этого прослушивание китайского возобновляется само.
3. Вся беседа сохраняется лентой (оригинал + перевод + время). Кнопка «Стенограмма» отправляет протокол переговоров в WhatsApp, Telegram, почту — куда угодно.

**Что ещё умеет:**

- Перевод работает **офлайн** (модели ML Kit скачиваются один раз при первом запуске) — критично для поездок в КНР, где сервисы Google недоступны.
- **Выбор языка собеседника** (шестерёнка справа вверху): путунхуа — стандартный китайский, тайваньский мандарин, кантонский (Гонконг/Гуандун). Текущий выбор виден чипом на главном экране и сохраняется между запусками.
- «Словарь терминов» — свои замены в переводе: марки стали, артикулы, Incoterms и прочие термины, которые машинный перевод коверкает.
- Полудуплекс против эха: пока телефон озвучивает перевод, микрофон молчит и не переводит сам себя.
- Собеседнику дают договорить: пауза в речи до ~1,2 секунды не обрывает фразу.
- Экран не гаснет, пока идёт сеанс перевода.
- Нажатие на любую реплику копирует её (оригинал + перевод) в буфер обмена.
- Живая строка распознавания — видно, что телефон слышит, ещё до конца фразы.
- Если офлайн-модели не скачались, на экране появится кнопка «Повторить загрузку моделей» — без перезапуска приложения.
- Статусы на экране: готовность моделей, наушники, выбранный язык.

## Диалекты и качество перевода — честно

- **«Все диалекты» не распознаёт ни одна система в мире** — ни это приложение, ни Timekettle за 65 тысяч рублей, ни Google. Хорошая новость: деловые переговоры в Китае ведутся на путунхуа (стандартном китайском) — это норма делового этикета, и путунхуа приложение понимает отлично.
- Кантонский (Гонконг, Гуандун) распознаётся при выборе в настройках, но перевод с него слабее: другая лексика и традиционная иероглифика. Региональные диалекты — шанхайский, сычуаньский, миньские — не поддерживаются нигде; если партнёр говорит на диалекте, вежливо попросите перейти на путунхуа, это обычная просьба.
- Офлайн-перевод пары китайский↔русский в ML Kit идёт через английский: смысл передаёт надёжно, формулировки упрощает. Абсолютный потолок качества сегодня — облачные движки (DeepL и подобные), но они платные, требуют интернет и не работают в КНР без VPN. Для стола переговоров надёжный офлайн ценнее.
- **Юридически значимые условия (цены, сроки, ответственность, штрафы) обязательно перепроверяйте с живым переводчиком.**

## Как получить установочный APK (≈10 минут, ничего устанавливать не нужно)

Проект уже содержит облачный сборщик: сервис GitHub Actions (бесплатный) сам скомпилирует APK на серверах GitHub. Ваши шаги:

1. **Зарегистрируйтесь на github.com** (если аккаунта нет). Обычная регистрация по почте, 2 минуты.
2. **Создайте репозиторий:** кнопка «+» справа вверху → *New repository* → имя, например `tolmach` → внизу *Create repository*.
3. **Загрузите файлы проекта:** на странице нового репозитория нажмите ссылку *uploading an existing file*. Распакуйте архив `Tolmach-android.zip` у себя на компьютере и перетащите в окно браузера **всё содержимое папки** (включая скрытую папку `.github` — она обязательна, именно в ней сборщик). Внизу нажмите *Commit changes*.
   - Если браузер не переносит папку `.github`: создайте файл вручную — *Add file → Create new file*, в имени введите `.github/workflows/build-apk.yml` и вставьте содержимое одноимённого файла из архива.
4. **Запустите сборку:** вкладка *Actions* сверху. При первом заходе GitHub спросит разрешение — нажмите *I understand my workflows, go ahead and enable them*. Сборка «Build APK» стартует сама после загрузки файлов; если нет — выберите её слева и нажмите *Run workflow*. Подождите 5–8 минут до зелёной галочки.
5. **Скачайте APK:** откройте завершившийся запуск → внизу блок *Artifacts* → скачайте `Tolmach-APK` (это zip, внутри `Tolmach-v1.1.0.apk`).
6. **Установите на телефон:** перекиньте APK на телефон (Telegram «Избранное» — самый простой способ), откройте файл, разрешите установку из неизвестных источников, когда Android спросит.

Если сборка завершилась красным крестом — откройте её, скопируйте текст ошибки и пришлите мне в чат: поправлю проект, вы обновите один файл и перезапустите сборку.

### Альтернатива: сборка на своём компьютере

Установите [Android Studio](https://developer.android.com/studio), откройте папку проекта (*Open*), дождитесь синхронизации (студия сама предложит создать Gradle wrapper — согласитесь) и выполните *Build → Build App Bundle(s) / APK(s) → Build APK(s)*.

## Первый запуск — обязательно дома, с интернетом

1. Откройте приложение и разрешите доступ к микрофону.
2. Дождитесь статуса «Модели готовы» — скачаются офлайн-модели перевода (~60–80 МБ, лучше по Wi-Fi). Если не вышло — кнопка «Повторить загрузку моделей».
3. Проверьте китайский голос: нажмите «Ответить по-русски», скажите «Здравствуйте, рад знакомству» — телефон должен произнести фразу по-китайски. Если молчит — в настройках Android → *Специальные возможности → Синтез речи* докачайте китайский голос Google.
4. **Перед поездкой в Китай:** там сервисы Google заблокированы, поэтому модели перевода и офлайн-пакеты распознавания речи (настройки Google → «Голосовой ввод» → «Распознавание речи офлайн» → китайский и русский) скачайте заранее.

## Советы для переговоров

- Телефон — на стол, микрофоном к собеседнику, экраном к себе.
- Просите партнёра говорить фразами по 1–2 предложения с паузами — так перевод точнее и быстрее.
- Если ваши наушники перехватывают микрофон и телефон стал хуже слышать партнёра, отключите для наушников профиль «Телефон/звонки» в настройках Bluetooth, оставив только «Медиа» — тогда слушать будет микрофон телефона, а перевод по-прежнему пойдёт в наушники.
- Заранее внесите в «Словарь» названия вашей продукции и термины сделки.
- Стенограмму отправляйте себе на почту сразу после встречи — это ваш протокол.

## Честные ограничения

- Распознавание китайской речи выполняет системный сервис Google: онлайн — отлично, офлайн-качество зависит от модели телефона.
- Направление китайского звука строго в динамик работает на Android 9+; на Android 8 звук идёт стандартным маршрутом.
- Задержка цикла «фраза → перевод в ухе» — примерно 1,5–3 секунды, это нормально.

## Техническая начинка

Kotlin 2.0, Jetpack Compose (Material 3), MVVM. Распознавание — системный `SpeechRecognizer` (zh-CN / zh-TW / yue-Hant-HK / ru-RU, частичные результаты, удлинённая пауза конца фразы). Перевод — ML Kit Translation, офлайн. Озвучка — системный `TextToSpeech`; китайские фразы синтезируются в файл и проигрываются через `MediaPlayer` с принудительной маршрутизацией на встроенный динамик (`setPreferredDevice`), русские идут обычным маршрутом в Bluetooth-наушники. minSdk 26 (Android 8), targetSdk 34.
