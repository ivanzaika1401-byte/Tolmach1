# Толмач × AGROPLANET — переводчик переговоров с Китаем

Фирменное Android-приложение для команды **ТОО «AGROPLANET» (Костанай)** — экспортёра зерновых и масличных культур. Живой перевод китайский ↔ русский на встречах с китайскими покупателями: телефон на столе, вы — в Bluetooth-наушниках.

**Три кнопки пульта:**

- **Зелёная сфера «Слушать китайский»** — партнёр говорит, перевод тихо звучит вам в наушники. Пока идёт запись, сфера пульсирует.
- **«Ответить»** — говорите по-русски, телефон произносит фразу по-китайски через динамик и сам возвращается к прослушиванию.
- **«Фразы» (золотая молния)** — разговорник Agroplanet: 18 готовых формулировок продавца зерна на выверенном деловом китайском, в одно касание, без задержки и ошибок распознавания. Знакомство компании, продукция (пшеница, ячмень, овёс, лён, подсолнечник, рапс, отруби), образцы и сертификаты (фитосанитарный, происхождения), цена DAP Алашанькоу/Хоргос, оплата предоплатой или аккредитивом, ж/д логистика, ход переговоров.

**Возможности:**

- Перевод полностью **офлайн** после первого запуска — критично для КНР, где сервисы Google недоступны.
- Лента переговоров: перевод крупно, оригинал под ним, время; нажатие копирует реплику, кнопка-динамик озвучивает её повторно (партнёр переспросил — повторили в одно касание).
- Строка сеанса: начало встречи и число реплик; фирменная стенограмма AGROPLANET — в WeChat, WhatsApp, почту из меню.
- Выбор языка собеседника: путунхуа / тайваньский мандарин / кантонский.
- Словарь терминов — свои замены в переводе (сорта, ГОСТы, базисы).
- Полудуплекс против эха; пауза до ~1,2 с не рвёт фразу собеседника; китайский синтез слегка замедлен для разборчивости.
- **Набор текстом** (клавиша ⌨ в шапке): напечатали по-русски — телефон перевёл и произнёс по-китайски. Спасение в шумном зале и для точных формулировок.
- **Свои фразы**: там же кнопка «В разговорник» — приложение само переводит вашу фразу и сохраняет в раздел «Мои фразы» с удалением; готовите набор под конкретную встречу заранее.
- **Протокол не теряется**: лента переговоров автоматически сохраняется и восстанавливается после закрытия приложения (до 200 последних реплик); «Очистить» стирает и её.
- Предупреждение о почти нулевой громкости медиа перед началом прослушивания.
- Пункт «О приложении» с версией — в меню.
- Умная защита: после трёх сбоев распознавания подряд прослушивание останавливается с понятным объяснением, а не молотит ошибками.
- Экран не гаснет во время сеанса; повторная загрузка моделей одной кнопкой.

**Дизайн** — собственный, с нуля (сайт компании сознательно не копировался): всегда тёмная тема, системные полосы Android в цвет приложения, фирменный шрифт Golos Text с полной кириллицей (SIL OFL, лицензия в комплекте), палитра «нефрит + пшеничное золото на графите» — нефрит как камень деловых подарков в Китае, золото как зерно и цвет достатка в китайской культуре. Иконка — колос-эквалайзер с золотыми зёрнами.

## Куда звучит каждый канал — полный контроль

Оба канала перевода направляются независимо: **русский** и **китайский** — каждый на любой из маршрутов (авто / Bluetooth / провод-USB / динамик). На главном экране под приборной панелью всегда видна строка маршрутов, например «Русский → Bluetooth · Китайский → провод/USB»; нажатие на неё (или на панель) открывает настройки. Там же — три готовые схемы одним касанием:

- **«Динамик для собеседника»** — классика: русский вам в наушники, китайский из динамика.
- **«Две пары наушников»** — русский в Bluetooth, китайский в проводные/USB.
- **«Наушник у собеседника»** — китайский перевод звучит в Bluetooth-наушнике у китайского партнёра, русский — из динамика вам. Схема для случая, когда наушники нужны именно собеседнику.

После выбора схемы нажмите «Проверить русский канал» и «Проверить китайский канал» — тестовые фразы подтвердят разводку за минуту до встречи.

## Две пары наушников — для русской и китайской стороны

Рабочая схема: **русская сторона — Bluetooth-наушники, китайская — проводные или USB-C наушники** (вторая пара подключается в разъём телефона).

1. Подключите обе пары — в приборной строке появится «Наушники: 2 пары».
2. Настройки (⋮ → Настройки): «Русский перевод» → *Bluetooth-наушники*, «Китайский перевод» → *Проводные / USB — вторая пара*.
3. Нажмите «Проверить русский канал» и «Проверить китайский канал» — тестовые фразы прозвучат каждая в своей паре. Это удобно сделать за минуту до встречи.

Честно о границах: **два Bluetooth с разным звуком Android не поддерживает** — система держит активным один BT-канал для медиа, это ограничение любых телефонов (Samsung Dual Audio лишь дублирует одинаковый звук в обе пары). Поэтому вторая пара — проводная или USB-C; для телефонов без разъёма подойдёт переходник USB-C → 3,5 мм. Вторую пару берите **без микрофона** (или простые вкладыши): гарнитура с микрофоном перехватит запись, и телефон будет слушать её микрофон вместо своего. Если выбранных наушников не оказалось, звук мягко уходит на динамик (китайский) или системный маршрут (русский) — встреча не сорвётся.

## Диалекты и качество — честно

Деловой Китай говорит на путунхуа — его приложение понимает на уровне лучших систем. Кантонский выбирается в настройках (перевод слабее). Региональные диалекты не поддерживает ни одна массовая система в мире — вежливо попросить путунхуа нормально. Офлайн-перевод zh↔ru идёт через английский: смысл надёжен, формулировки проще. **Цены, сроки и штрафы в контракте перепроверяйте с живым переводчиком.**

## Как получить установочный APK (≈10 минут, ничего не устанавливая)

В проект вшит облачный сборщик GitHub Actions.

1. Аккаунт на **github.com** → **New repository** → имя `tolmach` → *Create repository*.
2. *Uploading an existing file* → перетащите всё содержимое распакованного архива → *Commit changes*. Если папка `.github` не перенеслась — *Add file → Create new file*, имя `.github/workflows/build-apk.yml`, вставьте содержимое одноимённого файла из архива.
3. Вкладка **Actions** → разрешите workflows → сборка «Build APK» запустится сама (5–8 минут).
4. Готовый запуск → **Artifacts** → скачайте `Tolmach-APK` (внутри `Tolmach.apk`) → на телефон → установить, разрешив неизвестные источники.

### Уже загружали проект? Обновление за 3 минуты

**Add file → Upload files** → перетащите всё содержимое нового архива **кроме папки `.github`** (ваш сборщик подходит без правок) → *Commit changes* → Actions соберёт сам → установите новый APK поверх старого, словарь и настройки сохранятся.

Красный крест в Actions? Скопируйте текст ошибки и пришлите в чат — исправлю.

### Локальная сборка
Android Studio → *Open* → синхронизация → *Build → Build APK(s)*.

## Первый запуск — дома, с интернетом

1. Разрешите микрофон. 2. Дождитесь «Перевод: готов» (~60–80 МБ, Wi-Fi). 3. Проверьте китайский голос кнопкой «Фразы» → «Спасибо за встречу!» — телефон должен сказать это по-китайски; молчит — Android → Спец. возможности → Синтез речи → докачать китайский. 4. **Перед поездкой в КНР** скачайте офлайн-пакеты распознавания (настройки Google → Голосовой ввод → офлайн-языки: китайский, русский).

## Советы для переговоров

Телефон микрофоном к собеседнику. Просите говорить по 1–2 предложения. Наушники перехватили микрофон — отключите им профиль «Телефон/звонки», оставьте «Медиа». Термины — заранее в «Словарь», стенограмму — себе на почту сразу после встречи.

## Техническая начинка

Kotlin 2.0, Jetpack Compose (Material 3), MVVM, Golos Text 400/500/600/700. `SpeechRecognizer` (zh-CN / zh-TW / yue-Hant-HK / ru-RU, partial results, удлинённая пауза), ML Kit Translation офлайн, `TextToSpeech` + `MediaPlayer.setPreferredDevice` (независимая маршрутизация каждого канала: система / Bluetooth / провод-USB / динамик). minSdk 26, targetSdk 34, версия 2.3.0.
