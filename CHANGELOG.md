# Толмач — история версий

## 4.5.1 — полевой фикс первой облачной сборки 4.x
- Устранён дубликат ресурса color/ic_launcher_background (объявлен
  в colors.xml с 1.0 и повторно моим файлом в 4.2) — AAPT2 ронял
  assembleDebug за 46 секунд ещё до компиляции Kotlin.
- Контроль дубликатов имён ресурсов добавлен в постоянные проверки:
  этот слой вне досягаемости kotlinc-пирамиды.

## 4.5.0 — зеркальный режим
- Выбор китайского интерфейса переключает роль оператора: распознавание,
  направления перевода, тихий/громкий каналы, фразы, набор, повтор и свои
  фразы работают зеркально. Возобновление распознавания переведено на
  режимы вместо языковых тегов.
- Звуковые схемы стали ролевыми; первый выбор 中文 переворачивает каналы
  по умолчанию; 18 китайских строк переписаны под зеркальную роль;
  голосовая отправка доступна там, где перевод китайский. 57/57 проверок.

## 4.4.0 — двуязычный интерфейс
- Полная локализация RU / 中文（简体）: ~130 строк интерфейса в собственном
  слое AppStrings, деловой китайский выверен вручную.
- Выбор языка при первом входе; переключатель — первая секция Настроек;
  смена мгновенная, без перезапуска.
- Ошибки движка звонка переведены через систему ключей — мозг говорит
  на языке пользователя. Четыре новые поведенческие проверки; 54/54.

## 4.3.0 — открытие приложения
- Фирменная заставка при запуске: колос-эквалайзер на глубоком тёмном
  (androidx core-splashscreen, работает на всех поддерживаемых Android).
- Плавный вход интерфейса после заставки (растворение + мягкий подъезд).
- Краткий гид при первом запуске: «Слушать китайский / Ответить / Фразы
  и Звонок»; повторный вызов — Настройки → Инструменты. Три новые
  поведенческие проверки; пирамида: 50/50.

## 4.2.0 — лицо и словарь профессии
- ИСПРАВЛЕН баг со времён 2.2: свои фразы сохранялись с перепутанными
  полями (позиционные аргументы) — русский текст попадал в китайское поле
  и озвучивался партнёру китайским голосом. Пойман поведенческой
  проверкой №59 пирамиды; поля теперь именованы.
- Фирменная адаптивная иконка приложения: нефритовый колос-эквалайзер
  с золотым зерном на глубоком тёмном (включая монохром для Android 13+).
- Разговорник расширен до 40 фраз делового китайского зернотрейдера:
  параметры качества, условия оплаты, вагонная логистика через
  Алашанькоу/Хоргос, новая категория «Документы».

## 4.1.0 — дизайн и анимации
- Перекрёстное растворение между вкладками и между фазами звонка.
- Пульсирующие точки ожидания при подготовке кода и установке соединения.
- «Дышащая» иконка пустого экрана перевода.
- Активная звуковая схема в настройках подсвечивается (нефрит + индикатор).
- Тактильный отклик при копировании реплики из ленты.

## 4.0.5
- Верификационная пирамида встроена в облачный конвейер GitHub Actions:
  каждая сборка сначала проходит типовую компиляцию всего проекта и
  47 поведенческих проверок, затем JUnit, затем собирается APK.
- Повтор реплики (кнопка-динамик) в активном звонке уходит в канал собеседнику.
- CHANGELOG добавлен в репозиторий.

## 4.0.4
- Встроен публичный TURN-ретранслятор (Open Relay, три канала включая TCP-443):
  дожимает соединение через «симметричные» NAT; передаёт только зашифрованные
  DTLS-SRTP-пакеты. Тексты приложения уточнены: общая сеть не нужна,
  работает через мобильный интернет.

## 4.0.1–4.0.3 — инфраструктура качества
- Полный комплект Compose-стабов: весь проект (включая интерфейс) проходит
  типовую компиляцию без Android SDK; 0 ошибок.
- Живой smoke мозга: трамплин корутин, функциональный мини-JSON,
  детерминированный переводчик — 47 исполненных проверок, включая настоящий
  круг персистентности и полное рукопожатие защищённого звонка A↔B.
- CallEngine: статусы публикуются до асинхронной цепочки.

## 4.0.0 — интерфейс четырёх вкладок
- Нижняя навигация: Перевод · Фразы · Звонок · Настройки.
- Разговорник и настройки — полноценные экраны; из «Фраз» в активном звонке
  реплика уходит прямо в канал; золотая иконка звонка при связи.

## 3.9.х — полевые фиксы
- Спасение фразы из частичного результата (болячка Android: живой партиал,
  пустой финал) — найдено первым полевым тестом.
- Громкие сообщения вместо тихих обрывов: модели не скачаны, голос отсутствует.

## 3.7–3.8 — верификация без SDK
- Парсер-проверка всех исходников настоящим kotlinc; JUnit-тесты в CI;
  типовая компиляция движков против рукописных стабов.

## 3.4–3.6 — защищённый звонок
- P2P-аудиозвонок на WebRTC с обязательным шифрованием DTLS-SRTP,
  серверлесс-сигнализация обменом кодами; «Фразы» и «Написать» в звонок;
  копирование/вставка кодов, «Динамик/У уха», Back завершает звонок;
  найден и исправлен компилируемый баг хвостовой лямбды; экспорт .txt.

## 3.0–3.3 — движки и премиум-полировка

  «Вернуть» после очистки; стоп-кнопка озвучки; голосовые аудио в мессенджеры;
  «Проверка перед встречей».
