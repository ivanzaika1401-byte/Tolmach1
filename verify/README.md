# Верификация без Android SDK

`run.sh` — две ступени на любом kotlinc:

1. **Типы.** Движки (распознавание, перевод, озвучка, звонок) и мозг
   (TranslatorViewModel, ~750 строк) компилируются против рукописных
   заглушек в `stubs/` с точными сигнатурами внешних API. Ноль ошибок =
   каждый вызов, лямбда и suspend корректны.
2. **Поведение.** Мозг запускается по-настоящему: 26 живых проверок —
   маршруты, глоссарий, лента, стенограмма с брендом, экспорт .txt на диск,
   очистка и «Вернуть», громкость, звонок, и персистентность пяти настроек
   через пересоздание ViewModel (SharedPreferences в стабах — рабочие).

Стабы намеренно минимальны: новый вызов внешнего API уронит ступень 1,
и заглушка дополняется осознанно. Пройдено на Толмаче 3.9.0.
